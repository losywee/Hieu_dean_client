### Source Installation
Usage:

```
1. make

sudo make && sudo make install

2. Install

chmod +x Install.sh
sudo ./Install.sh

3. Run

Terminal:   jwwclient

```
### Binary Installation

Usage:
```
1. cd

cd package_linux_x64

2. Install

chmod +x INSTALL.sh
sudo ./INSTALL.sh

3. Run

You can find a shortcut in the start-menu.


```
